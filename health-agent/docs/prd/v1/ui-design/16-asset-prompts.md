# 素材生成 Prompt

> 本文件为"健康管家"产品所需的视觉素材提供 AI 图像生成 Prompt。
> 每个素材包含：素材名称、用途、尺寸、Midjourney/DALL-E/GPT Image Prompt（英文）、风格参考说明。
> 品牌色：珊瑚橙 #FF7A5C，背景 #F7F8FA，卡片白底 12px 圆角。

## 11. 图标方案（推荐使用开源图标库）

### 11.1 推荐方案：使用开源图标库

**推荐理由：**
- 质量高、风格统一
- React Native 有现成组件库（`react-native-vector-icons`）
- 不需要额外生成和导入成本
- 图标数量丰富，覆盖所有场景

**推荐图标库：**
- **Feather Icons** - 简洁、线性、现代（推荐）
- **Lucide** - Feather 的增强版，图标更多
- **Heroicons** - Tailwind 官方图标库

**React Native 集成：**
```bash
npm install react-native-vector-icons
```

### 11.2 Tab 导航图标映射表

| Tab 名称 | 图标库 | 图标名称 | 默认态颜色 | 选中态颜色 |
|---------|--------|---------|-----------|-----------|
| 首页 | Feather | `home` | #999999 | #FF7A5C |
| 饮食 | Feather | `coffee` 或 `utensils` | #999999 | #FF7A5C |
| 数据 | Feather | `bar-chart-2` | #999999 | #FF7A5C |
| 我的 | Feather | `user` | #999999 | #FF7A5C |

**使用示例：**
```jsx
import Icon from 'react-native-vector-icons/Feather';

<Icon name="home" size={24} color={isActive ? '#FF7A5C' : '#999999'} />
```

## 素材生成工作流建议

### 使用 Midjourney

1. 在 Discord 中使用 `/imagine` 命令，粘贴上述 Prompt
2. 生成 4 个变体后，选择最符合需求的版本
3. 使用 `U1-U4` 放大选中的图像
4. 如需微调，使用 `Vary (Strong)` 或 `Vary (Subtle)` 进一步优化
5. 下载高清图像，使用 Figma 或 Photoshop 去除背景（如需透明背景）

### 使用 DALL-E 3

1. 在 ChatGPT Plus 或 API 中使用上述 Prompt
2. 生成后如不满意，可要求调整具体元素（如颜色、角度、细节）

### 图标生成替代方案

对于 Tab 导航图标和功能图标，推荐直接使用开源图标库：

- **Feather Icons**：https://feathericons.com/
- **Lucide Icons**：https://lucide.dev/
- **Heroicons**：https://heroicons.com/
- **Iconify**：https://iconify.design/

这些图标库提供 SVG 格式，可直接在 Figma 中导入并调整线宽和颜色，确保视觉一致性。
