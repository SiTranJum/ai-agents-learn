# 身体数据编辑详情页

> 对应 PRD：`04-body-tracking.md`
> 设计规范：`01-design-system.md` · 组件库：`02-components.md`

---

## 1. 页面目标

提供表单化的身体数据编辑能力，让用户可以精确修改体重、围度、睡眠、运动等记录。

本页面是 AI 对话式记录的补充兜底：
- 当 AI 解析结果不够准确时，用户可以在这里手动修正。
- 当用户需要精确输入数值（如称重、计时）时，表单比对话更高效。
- 根据记录类型动态展示不同字段，一个页面覆盖所有身体数据编辑场景。
- 支持新增记录和编辑已有记录两种模式。

---

## 2. 页面入口

- 数据页待确认卡片点击 [✏️ 修改] 按钮。
- 数据页已记录卡片长按或点击后的"编辑"入口。
- 数据页未记录卡片点击 [+ 手动记录] 按钮。

## 3. 页面布局

### 3.1 整体布局说明

页面采用动态表单设计，根据记录类型（体重/围度/睡眠/运动）展示不同的字段组合。从上到下依次为：
1. 顶部导航栏：返回按钮 + 页面标题（根据记录类型动态变化）
2. 日期选择器：选择记录日期
3. 动态表单区：根据记录类型展示对应字段
4. 底部操作栏：保存按钮 + 取消按钮

### 3.6 布局细节说明

- 顶部导航栏固定，标题根据记录类型动态变化。
- 日期选择器默认为今天（体重/围度/运动）或昨天（睡眠）。
- 所有数值输入框点击后弹出数字键盘，支持小数输入。
- 时间选择器点击后弹出时间滚轮选择器（iOS 风格）。
- 睡眠时长根据入睡时间和起床时间自动计算，只读显示。
- 运动消耗热量根据运动类型和时长自动估算，只读显示。
- 底部操作栏固定在页面底部，保存和取消按钮并排。
- 围度字段均为可选，用户可以只填部分围度（如只量了腰围）。
- 运动类型选择器支持搜索和自定义输入，不限于预设列表。
- 卡片白底，12px 圆角，与全局设计规范一致。

### 4.1 重点组件说明

- **DatePicker**：日期选择器，点击弹出日历面板。默认值根据记录类型决定（睡眠默认昨天，其他默认今天）。不允许选择未来日期。
- **WeightInput**：数字输入框，支持一位小数（如 74.5），点击弹出数字键盘。输入范围 20.0~300.0 kg。
- **WaistInput / HipInput / ThighInput / UpperArmInput**：围度输入框，均为可选字段，支持一位小数，单位 cm。
- **SleepTimePicker / WakeTimePicker**：时间选择器，滚轮式选择，精确到分钟。入睡时间支持跨天（如 23:30 到次日 07:00）。
- **DurationDisplay**：只读文本，灰色字体，根据入睡和起床时间自动计算。
- **QualitySelector**：三个选项的单选按钮组，选中项高亮（珊瑚橙 #FF7A5C 背景）。
- **ExerciseTypeSelector**：下拉选择器，预设常见运动类型，支持搜索和自定义输入。
- **DurationInput**：数字输入框，整数，单位分钟。
- **CalorieDisplay**：只读文本，灰色字体，根据运动类型和时长自动估算。
- **SaveButton**：珊瑚橙 #FF7A5C 填充，白色文字，圆角 12px。
- **CancelButton**：白色背景，灰色边框，灰色文字，圆角 12px。

### 5.3 表单校验失败

适用场景：用户点击保存但存在必填项未填或数据异常。

#### 校验规则

| 记录类型 | 必填字段 | 校验规则 |
|---------|---------|---------|
| 体重 | 体重值 | 必须大于 0，范围 20.0~300.0 kg |
| 围度 | 至少填一项 | 每项填写时必须大于 0 |
| 睡眠 | 入睡时间、起床时间 | 起床时间必须晚于入睡时间 |
| 运动 | 运动类型、时长 | 时长必须大于 0 |

#### 错误提示方式
- 对应输入框下方显示红色错误提示文字（font-caption，#FF4D4F）。
- 输入框边框变为红色。
- 点击保存后自动滚动到第一个错误位置。

## 6. 关键交互

### 6.1 选择日期 → 更新记录日期
- 点击日期选择器，弹出日历面板。
- 不允许选择未来日期，未来日期置灰不可点击。
- 选择日期后，日历面板收起，日期更新。
- 编辑模式下日期锁定不可修改。

### 6.2 输入体重 / 围度 → 数字键盘
- 点击数值输入框，弹出数字键盘（含小数点）。
- 支持一位小数输入。
- 输入完成后点击键盘"完成"按钮收起键盘。
- 输入值超出合理范围时，输入框下方显示警告提示。

### 6.3 选择入睡/起床时间 → 自动计算睡眠时长
- 点击时间选择器，弹出时间滚轮。
- 入睡时间和起床时间任一变化后，自动重新计算睡眠时长。
- 睡眠时长以"X小时Y分"格式显示。
- 支持跨天计算（如 23:30 → 07:00 = 7小时30分）。

### 6.4 选择运动类型 / 输入时长 → 自动估算消耗
- 选择运动类型后，系统根据类型和时长自动估算热量消耗。
- 修改时长后实时重新计算。
- 消耗值为估算值，显示"~"前缀（如"~280 kcal"）。
- 自定义运动类型时，消耗显示"--"（无法估算）。

### 6.5 保存 → 返回数据页，卡片更新
- 点击保存按钮。
- 系统执行表单校验。
- 校验通过后保存数据。
- 返回数据页。
- 对应 Tab 的今日记录卡片更新为最新内容。
- 趋势图数据同步刷新。
- 底部弹出 Toast 提示"已保存体重记录"（根据类型动态变化）。

### 6.6 取消 → 返回，不保存
- 点击取消按钮或顶部返回按钮。
- 如果表单有未保存的修改，弹出确认对话框："当前修改尚未保存，确认离开吗？"
- 用户确认后返回数据页，不保存任何变更。
- 如果表单无修改，直接返回。

### 6.7 睡眠质量选择
- 三个选项横向排列：好 / 一般 / 差。
- 点击选中，选中项背景变为珊瑚橙 #FF7A5C，文字变白。
- 未选中项为灰色边框、灰色文字。
- 默认无选中，用户必须主动选择。
