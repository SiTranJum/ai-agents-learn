# 数据页

> 对应 PRD：`04-body-tracking.md`
> 设计规范：`01-design-system.md` · 组件库：`02-components.md`

---

## 1. 页面目标

展示体重、围度、睡眠、运动、饮水、排便等身体数据的记录与趋势，支持 AI 对话式记录和修改，帮助用户直观了解身体变化。

---

## 2. 页面入口

- 底部 Tab 导航 "数据"（第 3 个 Tab）
- 首页辅助记录小卡片点击跳转

---

## 5. 页面状态

每个 Tab 都有三种状态：

### 5.1 empty（未记录）

今日无记录，卡片显示引导文案。

```
今日体重
┌─────────────────────────────────┐
│                                 │
│     还没有记录今天的体重          │
│     告诉 AI "今天 74.5kg"       │
│                                 │
│         [+ 手动记录]             │
│                                 │
└─────────────────────────────────┘
```

### 5.2 pending（待确认）

AI 解析出数据，等待用户确认。卡片边框高亮（`brand-primary-light` 背景）。

**体重 Tab 待确认状态：**
