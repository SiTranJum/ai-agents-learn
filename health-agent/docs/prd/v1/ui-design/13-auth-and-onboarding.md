## 页面 A：登录页

### A1. 页面目标

为已注册用户提供登录入口，通过邮箱 + 密码方式完成身份验证。页面风格温暖亲切，传达"健康管家是你的朋友"的品牌调性。

### A2. 页面入口

- App 启动时未登录状态默认进入
- 注册页底部"已有账号？去登录"链接
- 退出登录后跳转至此

### A5. 页面状态

#### A5.1 默认态

表单为空，登录按钮为禁用态（浅灰色），输入内容后按钮激活为品牌色。

#### A5.2 输入中态

| 区域 | 展示差异 |
|------|---------|
| 当前输入框 | 边框变为品牌色 `#FF7A5C`，左侧图标变为品牌色 |
| 键盘 | 弹起键盘，页面内容上推避免遮挡 |

#### A5.3 登录中态

| 区域 | 展示差异 |
|------|---------|
| 登录按钮 | 显示 loading 动画 + "登录中..."，不可重复点击 |
| 表单 | 输入框变为不可编辑 |

#### A5.4 登录失败态

| 区域 | 展示差异 |
|------|---------|
| 表单上方 | 显示红色错误提示条："邮箱或密码错误，请重试" |
| 密码输入框 | 清空密码内容，自动聚焦 |

### A6. 关键交互

| 操作 | 行为 |
|------|------|
| 输入邮箱和密码 | 两个字段都有内容时，登录按钮从禁用态变为可用态 |
| 点击密码显示/隐藏 | 切换密码明文/密文显示，图标在睁眼/闭眼之间切换 |
| 点击"登录" | 校验邮箱格式 → 调用登录 API → 成功跳转首页 / 失败显示错误 |
| 点击"忘记密码？" | 跳转到忘记密码页（页面 C） |
| 点击"去注册" | 跳转到注册页（页面 B） |
| 键盘回车 | 邮箱框回车跳到密码框，密码框回车触发登录 |

### A8. Figma AI 生成描述

设计一个移动端 App 的登录页面，温暖亲切的健康生活方式风格，不要科技感。页面背景为纯白色或极浅灰（#FAFAFA），整体简洁大方。

页面顶部大留白后，居中展示一个 64px 的品牌 Logo（圆角方形，珊瑚橙渐变背景 + 白色健康相关图形），Logo 下方是品牌名"健康管家"（珊瑚橙色，16px）。

中间区域左对齐展示主标题"欢迎回来"（24px，深色加粗）和副标题"像朋友一样，陪你管理健康"（14px，灰色）。

下方是两个输入框，白底圆角 12px，带 1px 浅灰边框。邮箱输入框左侧有灰色邮箱线性图标，密码输入框左侧有灰色锁线性图标、右侧有眼睛图标用于切换显示。输入框聚焦时边框变为珊瑚橙色。

密码框下方右对齐"忘记密码？"灰色链接。再下方是全宽登录按钮，珊瑚橙色背景（#FF7A5C），白色文字"登录"，高度 48px，圆角 24px。

页面底部居中"还没有账号？去注册"，其中"去注册"为珊瑚橙色可点击文字。不展示第三方登录按钮。整体干净温暖，适配 iPhone 14/15 尺寸，中文界面。

### B8. Figma AI 生成描述

设计一个移动端 App 的注册页面，风格延续登录页，温暖、简洁、易理解。页面背景为白色或极浅灰（#FAFAFA），主色珊瑚橙（#FF7A5C）。

顶部左侧有一个轻量返回箭头。页面主体左对齐展示标题"创建你的健康档案"，标题下不需要额外副标题。下方是三个白底圆角输入框：邮箱、密码、确认密码。输入框圆角 12px，带浅灰边框，聚焦时变为珊瑚橙边框。密码和确认密码右侧都有眼睛图标。密码框下方可出现一行小字提示密码强度或规则。

标题下方与输入框之间留出舒适空白。输入框下方是一个全宽品牌色按钮，文案"注册"，高度 48px，圆角 24px。页面底部居中显示"已有账号？去登录"，其中"去登录"为珊瑚橙色。整体让用户感受到注册不是复杂开户，而是在创建自己的健康档案。适配 iPhone 14/15 尺寸，中文界面。

### D8. Figma AI 生成描述

设计一个移动端 App 的分步档案填写页面，引导式、友好、低压力的风格。页面背景白色或极浅灰（#FAFAFA），主色珊瑚橙（#FF7A5C）。

顶部有一个 5 段进度条，已完成段为珊瑚橙填充，当前段为珊瑚橙，未完成段为浅灰色。进度条下方显示"第 N 步，共 5 步"灰色小字。

每一步的标题区域左对齐，主标题 20px 加粗深色，副标题 14px 灰色。表单区域使用选择卡片（圆角 12px 白底卡片，选中时边框珊瑚橙 + 浅橙背景 #FFE7E1）代替传统下拉框，让选择更直观。数字输入使用滑块组件（轨道灰色，滑块珊瑚橙圆形），滑块上方实时显示数值。多选标签使用胶囊样式，未选中灰底灰字，选中珊瑚橙浅底 + 珊瑚橙文字。

底部固定操作栏，左侧"上一步"（灰色文字按钮），中间"跳过"（灰色文字按钮），右侧"下一步"（珊瑚橙填充按钮，白色文字，圆角 24px）。步骤切换有左右滑动动画。

完成后的预览页居中展示绿色勾选图标 + "档案已完成！"标题，下方是白底圆角卡片展示所有填写信息的摘要，底部是全宽珊瑚橙按钮"开始使用健康管家"和灰色"返回修改"链接。整体引导感强，让用户感觉在和 AI 对话建立档案而非填写枯燥表单。适配 iPhone 14/15 尺寸，中文界面。
