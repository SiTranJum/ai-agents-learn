# 课程 15：什么是 RAG？为什么 Agent 需要外部知识

## 学习目标

1. 理解 RAG 的核心原理和解决的问题
2. 理解 Embedding（向量化）的概念
3. 掌握 RAG 的完整流程
4. 理解 RAG 在健康管家中的应用场景

## 知识点

### 1. LLM 的知识局限
- 训练数据有截止日期
- 不知道你的私有数据
- 会编造不存在的信息（幻觉）

### 2. RAG 核心原理
- Retrieval（检索）+ Augmented（增强）+ Generation（生成）
- 先检索相关知识，再让 LLM 基于这些知识回答

### 3. Embedding（向量化）
- 把文本变成数字向量
- 语义相近的文本，向量也相近
- 向量数据库的作用

### 4. RAG 完整流程
- 离线：文档分块 → Embedding → 存入向量数据库
- 在线：用户提问 → Embedding → 向量检索 → 拼入 Prompt → LLM 回答

### 5. 健康管家的 RAG 场景
- 食物营养知识库
- 健康建议知识库
- 用户历史记忆检索

## 预计时长

60-90 分钟
