# 课程 12：Chain of Thought — 让 Agent 展示思考过程

## 学习目标
1. 理解 CoT（Chain of Thought）的核心原理
2. 掌握 CoT 的三种用法：Zero-shot CoT、Few-shot CoT、Self-Consistency CoT
3. 理解 CoT 与 ReAct 的关系和区别
4. 在健康管家场景中应用 CoT 提升推理质量

## 为什么这节课重要？

LLM 在处理复杂推理任务时容易"跳步"或出错。

**问题**：
```
用户：我今天吃了 2 个鸡蛋和一碗米饭，我的目标是每天 1500 卡，还能吃多少？

LLM 直接回答：你还能吃 1200 卡（❌ 算错了）
```

**CoT 的解决方案**：让 LLM 一步步推理
```
第一步：2 个鸡蛋 = 2 × 70 = 140 卡
第二步：1 碗米饭 = 200 卡
第三步：已摄入 = 140 + 200 = 340 卡
第四步：剩余 = 1500 - 340 = 1160 卡
答案：你还能吃 1160 卡路里
```
