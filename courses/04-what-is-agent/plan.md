# 课程 4：什么是 AI Agent？

## 学习目标
1. 理解 AI Agent 的定义和本质
2. 区分普通 LLM 调用和 Agent 的核心差异
3. 理解 Agent 的四大核心能力及其关系
4. 通过代码对比感受 Agent 和普通 LLM 调用的区别

## 为什么这节课重要？
前三节课我们学了 LLM、Prompt、Tool Use，这些都是"零件"。
这节课把它们组装起来，理解什么是 Agent。

## 课程内容

### 1. 从对比开始：普通 LLM 调用 vs Agent（20 分钟）
- 普通 LLM 调用：你问一句，它答一句
- Agent：能自主决策、多步执行、使用工具、保持记忆

### 2. Agent 的定义和本质（20 分钟）
- 业界定义
- 一句话理解 Agent
- Agent = LLM + 感知 + 记忆 + 工具 + 规划

### 3. 四大核心能力详解（30 分钟）
- 感知（Perception）：理解用户意图
- 记忆（Memory）：记住上下文和历史
- 工具（Tool Use）：与外部世界交互
- 规划（Planning）：拆解和编排任务

### 4. 代码对比：普通 LLM vs Agent（30 分钟）
- 用同一个场景对比两种实现
- 感受 Agent 的自主决策能力

## 实践任务
- 将普通 LLM 对话升级为简单 Agent
- 体验 Agent 的自主循环

## 预计时长
- 讲解 + 演示：约 60 分钟
- 练习 + 讨论：约 40 分钟
