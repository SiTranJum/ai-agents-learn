# 课程 17：构建完整的 RAG Pipeline

## 学习目标

1. 掌握文档加载和预处理
2. 理解文档分块（Chunking）策略
3. 实现完整的 RAG 系统（从文档到问答）
4. 优化检索质量

## 知识点

### 1. 文档加载
- 支持多种格式（PDF、Word、Markdown、TXT）
- 文本提取和清洗
- 元数据提取

### 2. 文档分块（Chunking）
- 为什么要分块
- 分块策略：固定长度、语义分块、递归分块
- chunk_size 和 chunk_overlap 的选择

### 3. 元数据管理
- 为每个 chunk 添加元数据（来源、章节、时间）
- 元数据在检索中的作用
- 混合检索：向量 + 元数据过滤

### 4. 检索优化
- 重排序（Reranking）
- 混合检索（向量 + 关键词）
- 查询改写

### 5. 实践案例
- 加载健康知识文档
- 智能分块
- 构建知识库
- 实现问答

## 预计时长

90-120 分钟
