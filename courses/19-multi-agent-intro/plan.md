# 课程 19：Multi-Agent 入门 — 为什么需要多个 Agent？

## 学习目标

1. 理解单 Agent 的局限性，知道什么时候需要 Multi-Agent
2. 掌握 Multi-Agent 的三种核心协作模式
3. 动手实现一个简单的 Multi-Agent 系统
4. 为健康管家产品的 Multi-Agent 架构打基础

## 前置知识

- 课程 4-8：Agent 基础（循环、记忆、规划、工具调用）
- 课程 11：ReAct 模式
- 课程 13：Multi-Step Agent

## 知识点

### 概念部分（30 分钟）
- 单 Agent 的天花板在哪里
- Multi-Agent 的核心思想：分工 + 协作
- 三种协作模式：串行管道、并行扇出、路由分发
- 健康管家场景下的 Multi-Agent 设计

### 实践部分（60 分钟）
- 实现串行管道：饮食记录 → 营养分析 → 建议生成
- 实现并行扇出：同时分析营养 + 运动 + 睡眠
- 实现路由分发：根据用户意图分发到不同 Agent

## 预计时长

90 分钟
