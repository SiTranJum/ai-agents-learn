# 课程 7：Planning（规划能力）

## 学习目标
1. 理解为什么 Agent 需要规划能力
2. 掌握任务分解的方法
3. 实现基础的规划系统
4. 理解规划在健康管家中的应用

## 为什么这节课重要？
前面学了：
- 课程 4：Agent 是什么
- 课程 5：Agent 循环（感知 → 思考 → 行动）
- 课程 6：Memory（记忆系统）

现在学 Planning：**让 Agent 能处理复杂的多步任务**。

## 课程内容

### 1. 为什么需要规划（15 分钟）
- 简单任务 vs 复杂任务
- 一步完成 vs 多步分解
- 规划的价值

### 2. 规划的两种方式（30 分钟）
- 方式 1：让 LLM 自己规划（ReAct 模式）
- 方式 2：预先定义计划模板

### 3. 实现规划系统（40 分钟）
- 任务分解
- 步骤执行
- 进度追踪

### 4. 健康管家的规划场景（15 分钟）
- 制定减肥计划
- 生成周报
- 多目标协调

## 预计时长
- 讲解 + 演示：约 60 分钟
- 练习：约 30 分钟
