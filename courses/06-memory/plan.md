# 课程 6：Memory（记忆系统）

## 学习目标
1. 理解为什么 Agent 需要记忆
2. 掌握三种记忆类型：短期、中期、长期
3. 实现基础的记忆管理系统
4. 理解记忆在健康管家中的应用

## 为什么这节课重要？
没有记忆的 Agent 就像失忆症患者，每次对话都是全新的。
记忆让 Agent 能够：
- 记住用户的偏好和习惯
- 理解上下文和历史
- 提供个性化的建议

## 课程内容

### 1. 为什么需要记忆（15 分钟）
- LLM 本身无记忆
- 对话历史 vs 长期记忆
- 记忆的价值

### 2. 三种记忆类型（30 分钟）
- 短期记忆：当前对话
- 中期记忆：最近几天的活动
- 长期记忆：用户画像、习惯、偏好

### 3. 实现记忆系统（40 分钟）
- 对话历史管理
- 结构化存储
- 记忆召回机制

### 4. 健康管家的记忆设计（15 分钟）
- 用户偏好记忆
- 饮食习惯记忆
- 目标和计划记忆

## 预计时长
- 讲解 + 演示：约 60 分钟
- 练习：约 30 分钟
