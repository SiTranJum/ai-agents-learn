# 课程 5：Agent 的核心循环 — 感知、思考、行动

## 学习目标
1. 理解 Agent 的核心循环：感知 → 思考 → 行动 → 观察 → 循环
2. 掌握如何实现一个基础的 Agent 循环
3. 理解循环的终止条件和错误处理
4. 用 Spring MVC 类比理解 Agent 循环

## 为什么这节课重要？
课程 4 讲了 Agent 是什么，这节课讲 Agent 怎么运转。
理解这个循环，就理解了 Agent 的核心机制。

## 课程内容

### 1. Agent 循环的本质（15 分钟）
- 什么是"循环"
- 为什么需要循环
- 循环的四个阶段

### 2. 用 Spring MVC 类比（15 分钟）
- Request → Controller → Service → Response
- 感知 → 思考 → 行动 → 观察

### 3. 实现一个基础 Agent 循环（40 分钟）
- 循环框架代码
- 工具注册和调用
- 终止条件判断

### 4. 错误处理和优化（20 分钟）
- 最大循环次数限制
- 工具调用失败处理
- 无限循环检测

## 预计时长
- 讲解 + 演示：约 60 分钟
- 练习：约 30 分钟
