## 学习流程

### 第一步：知识回顾（60 分钟）

阅读 `slides.md`，回顾课程 1-8 的核心知识点：

1. **LLM 基础**：Token、Temperature、上下文窗口、多轮对话
2. **Prompt Engineering**：五大原则、System Prompt
3. **Tool Use**：Function Calling 原理、工具定义
4. **Agent 概念**：Agent vs LLM、自主性、工具使用
5. **Agent Loop**：感知 → 思考 → 行动
6. **Memory**：短期、中期、长期记忆
7. **Planning**：ReAct、CoT、分步执行
8. **完整 Agent**：组件整合、架构设计

**学习建议**：
- 不要死记硬背，重点理解概念和原理
- 对照之前的课程资料，查漏补缺
- 思考各知识点之间的关系

### 第二步：完成测验（90 分钟）

打开 `quiz.md`，完成测验题目：

**理论题（50 分）**：
- 5 道选择题（25 分）
- 3 道判断题（15 分）
- 1 道简答题（10 分）

**实践题（50 分）**：
- 编写 Prompt（15 分）
- 定义工具（15 分）
- 实现简单 Agent（20 分）

**测验说明**：
- 开卷测验，可以查阅资料
- 重点考察理解和动手能力
- 不限时间，理解比速度重要

### 第三步：对照答案（30 分钟）

测验完成后，点击题目下方的"点击查看答案"，对照答案：

1. **检查答案**：看看自己答对了多少
2. **理解错误**：错误的题目，理解为什么错
3. **查阅资料**：不理解的知识点，回到相关课程复习
4. **总结薄弱点**：记录自己的薄弱环节

### 第四步：实践验证（60 分钟）

完成实践题的代码，运行测试：

**实践题 3：实现简单 Agent**

1. 将参考答案保存为 `simple_agent.py`
2. 设置环境变量：`export DEEPSEEK_API_KEY="your-key"`
3. 运行测试：`python simple_agent.py`
4. 测试对话：
   ```
   你：我今天早上吃了两个鸡蛋，140 卡路里
   AI：好的，已记录：鸡蛋，140 卡路里

   你：我今天总共摄入了多少热量？
   AI：您今天总共摄入了 140 卡路里
   ```

**扩展练习**：
- 添加更多工具（如查询营养信息）
- 优化 Prompt（让回复更自然）
- 添加记忆系统（持久化存储）

## 评分标准

| 分数段 | 评价 | 建议 |
|--------|------|------|
| 90-100 | 优秀 | 基础扎实，可以进入阶段三 |
| 75-89 | 良好 | 部分知识需要巩固，建议复习薄弱环节 |
| 60-74 | 及格 | 基础尚可，建议重点复习错误的题目 |
| <60 | 不及格 | 需要重新学习基础课程 |

**重要提示**：
- 分数不是目的，理解知识才是关键
- 错误是学习的机会，不要气馁
- 动手实践比理论分数更重要

## 常见问题

### Q1: 测验没考好怎么办？

**A**: 
- 不要气馁，这是学习过程的一部分
- 重点关注错误的题目，理解为什么错
- 回到相关课程，重新学习薄弱环节
- 可以重新做一遍测验，检验学习效果

### Q2: 实践题不会做怎么办？

**A**: 
- 先查看参考答案，理解代码逻辑
- 对照课程 8 的代码，找出相似之处
- 尝试修改参考答案，添加新功能
- 遇到问题，查阅 SDK 文档或课程资料

### Q3: 什么时候可以进入阶段三？

**A**: 
- 测验分数 ≥ 60 分
- 理解所有知识点（即使有些题做错了）
- 能独立实现简单的 Agent
- 对进阶内容有学习兴趣

### Q4: 阶段三会学什么？

**A**: 
- **高级 Agent 模式**：ReAct、CoT、Multi-Step
- **RAG**：检索增强生成，让 Agent 拥有外部知识
- **Multi-Agent**：多个 Agent 协作
- **框架选型**：LangChain、CrewAI 等
