# 课程 20：Agent 间通信与编排

## 学习目标

1. 理解 Multi-Agent 系统中的通信机制
2. 掌握三种消息传递模式：直接调用、消息队列、共享状态
3. 学会设计 Agent 间的协调机制
4. 为课程 21 的完整系统打基础

## 前置知识

- 课程 19：Multi-Agent 三种协作模式（串行、并行、路由）

## 知识点

### 概念部分（30 分钟）
- Agent 间通信的三种方式
- 消息格式设计：结构化 vs 自然语言
- 编排器（Orchestrator）的职责
- 状态管理：谁来记录执行进度

### 实践部分（60 分钟）
- 实现基于消息的 Agent 通信
- 实现一个简单的编排器
- 实现共享上下文的 Agent 协作

## 预计时长

90 分钟
