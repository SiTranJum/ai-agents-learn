# 课程 8：动手做一个完整的 Agent

## 学习目标
1. 整合课程 4-7 学到的所有组件
2. 构建一个功能完整的健康管家 Agent
3. 理解各组件如何协同工作
4. 为后续的健康管家项目打基础

## 为什么这节课重要？
前面学了 Agent 的各个组件：
- 课程 4：Agent 概念
- 课程 5：Agent 循环
- 课程 6：Memory（记忆系统）
- 课程 7：Planning（规划能力）

现在把它们**整合起来**，做一个真正能用的 Agent。

## 课程内容

### 1. 回顾组件（10 分钟）
- Agent 的四大组件
- 它们如何协同工作

### 2. 设计 Agent 架构（20 分钟）
- 整体架构设计
- 组件之间的关系
- 数据流向

### 3. 实现完整 Agent（60 分钟）
- Agent 核心类
- 整合 Memory
- 整合 Planning
- 整合 Tool Use

### 4. 测试和优化（20 分钟）
- 多轮对话测试
- 记忆召回测试
- 规划执行测试

## 实践任务
构建一个健康管家 Agent，支持：
- 记录饮食
- 查询营养信息
- 制定减肥计划
- 生成周报

## 预计时长
- 讲解 + 演示：约 60 分钟
- 实践：约 60 分钟
