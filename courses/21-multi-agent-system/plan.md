# 课程 21：构建一个简单的 Multi-Agent 系统

## 学习目标

1. 整合课程 19-20 学到的所有概念，构建一个完整的 Multi-Agent 系统
2. 实现健康管家的 Multi-Agent 架构：路由 + 管道 + 并行 + 编排器 + 共享上下文
3. 理解 Multi-Agent 系统的调试和可观测性
4. 为后续框架学习（课程 22+）打下实践基础

## 前置知识

- 课程 19：三种协作模式（串行、并行、路由）
- 课程 20：通信方式、编排器、共享上下文

## 知识点

### 概念部分（20 分钟）
- 系统架构设计回顾
- 可观测性：日志、执行追踪
- 错误处理策略

### 实践部分（70 分钟）
- 构建完整的健康管家 Multi-Agent 系统
- 支持：饮食记录（管道）、综合分析（并行）、知识问答、闲聊
- 加入执行追踪和日志

## 预计时长

90 分钟
