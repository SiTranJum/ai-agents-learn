# 课程 16：Embedding 与向量数据库入门（Milvus 实战）

## 学习目标

1. 理解真正的 Embedding API 如何使用
2. 掌握 Milvus 向量数据库的基本操作
3. 实现一个完整的 RAG 系统（真实 Embedding + Milvus）
4. 理解向量检索的性能优化

## 知识点

### 1. Embedding API
- OpenAI 的 text-embedding-3-small/large
- 国内替代方案（智谱、通义、百川）
- Embedding 模型的选择标准

### 2. Milvus 基础
- 安装和启动（Docker）
- Collection（集合）的概念
- 插入向量、搜索向量
- 索引类型和性能优化

### 3. 完整 RAG Pipeline
- 文档加载和分块
- 批量 Embedding
- 存入 Milvus
- 检索和问答

### 4. 实践案例
- 构建食物营养知识库
- 实现健康问答系统

## 预计时长

90-120 分钟（包含 Milvus 安装）
